Interactive data visualization

The assignment was to visualize data in an interactive environment. I found data on California grapes. I split the information into four pages: grape production, wine production, wine exports, and grape exports. I drew charts to visualize each piece of data (e.g., for the wine production chart, the size of the wine glass corresponds with the amount of wine produced that year), and I created icons that link to each page.
